# Visión del producto
Este proyecto es una **plataforma de ecommerce genérica**, diseñada para servir como base sólida y reutilizable para la venta de productos en línea. Su propósito es ofrecer una solución clara, modular y extensible, orientada principalmente a pequeños y medianos negocios que requieren una presencia digital eficiente sin complejidad innecesaria.

El producto busca cubrir los elementos esenciales de un ecommerce moderno: catálogo de productos, gestión de pedidos, flujo de compra y administración básica, manteniendo un enfoque en la simplicidad operativa y la claridad del dominio del negocio.

La plataforma está pensada para evolucionar de forma incremental, comenzando con un conjunto mínimo de funcionalidades bien definidas y permitiendo la incorporación progresiva de nuevas capacidades conforme crezcan las necesidades del negocio o del mercado.

Este ecommerce se concibe como una **base adaptable**, capaz de personalizarse según distintos contextos comerciales, manteniendo una separación clara entre reglas de negocio, lógica de aplicación e interfaz de usuario.

## Objetivos del producto
- Proveer una base funcional y mantenible para la venta de productos en línea.
- Facilitar la adaptación del sistema a distintos tipos de negocios.
- Priorizar una estructura clara que permita escalar el producto sin rehacerlo.
- Mantener un enfoque pragmático: resolver primero lo esencial, luego extender.

## Principios
- Simplicidad antes que complejidad innecesaria.
- Enfoque en un MVP funcional y coherente.
- Separación clara de responsabilidades.
- Evolución progresiva y controlada del producto.
